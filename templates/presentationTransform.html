Configure Presentation Transform

Presentation transform will wipe all data across all services and sample data will be created for presentation purposes.

The user account email MUST BE A GOOGLE EMAIL as it will be created as a Google OAuth Login account. It cannot be signed into via the standard login flow.

A sample forum post will be created and attached to the user account.

A sample support query will be created with the given query sender email. Customer support replies will be sent to this email address.
